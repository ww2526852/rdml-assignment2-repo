## Assessment 2 - RMDL

This repository contains the data, code, and report for Assessment 2 of the RMDL course.

### Contents

-   `data/`: contains the raw data file downloaded from Testable and the model cache

    -   `661427_260428_102656_long.csv`: raw results file containing all participants' trial data
    -   `cache/`: cached Bayesian model objects fitted with `brms`

-   `RMDL--StroopTask--TrialTemplate.csv` is the trial file

-   `report.qmd` is the Quarto analysis file

-   `report.pdf` is the rendered report
